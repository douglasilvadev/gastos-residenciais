# Sistema de Controle de Gastos Residenciais

Projeto full stack para controle de gastos residenciais, com separação entre back-end e front-end.

## Tecnologias

### Back-end
- C#
- .NET 8 Web API
- Entity Framework Core
- PostgreSQL

### Front-end
- React
- TypeScript
- Vite

## Objetivo
Permitir o cadastro de pessoas, categorias e transações financeiras, além da geração de relatórios com totais por pessoa e categoria.
